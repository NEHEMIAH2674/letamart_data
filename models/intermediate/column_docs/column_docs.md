{% docs basket_value_gbp %}
Total value of the order basket in GBP after line-level discounts.
Calculated as sum of (quantity × unit_price - discount) across all order lines.
{% enddocs %}

{% docs gross_basket_gbp %}
Total gross value of the order basket before any discounts are applied.
{% enddocs %}

{% docs total_discount_gbp %}
Total discount amount applied across all lines in the order, in GBP.
{% enddocs %}

{% docs item_line_count %}
Number of distinct product lines in the order.
{% enddocs %}

{% docs total_units %}
Total number of units ordered across all product lines.
{% enddocs %}

{% docs substituted_line_count %}
Number of order lines where the original product was substituted
with an alternative item during picking.
{% enddocs %}

{% docs has_promo %}
Boolean flag indicating whether a valid promotional code was
applied to this order.
{% enddocs %}

{% docs is_delivered %}
Boolean flag indicating whether the order reached delivered status.
Only delivered orders are counted in revenue and LTV calculations.
{% enddocs %}

{% docs is_cancelled %}
Boolean flag indicating whether the order was cancelled.
{% enddocs %}

{% docs is_returned %}
Boolean flag indicating whether the order was returned after delivery.
{% enddocs %}

{% docs order_date %}
Date the order was placed, extracted from order_created_at timestamp.
Used as the partition key in downstream fact tables.
{% enddocs %}

{% docs lifetime_value_gbp %}
Sum of basket values across all delivered orders for a customer, in GBP.
Only includes delivered orders — cancelled and returned orders are excluded.
{% enddocs %}

{% docs avg_order_value_gbp %}
Average basket value across all delivered orders for a customer, in GBP.
{% enddocs %}

{% docs days_since_last_order %}
Number of days between the customer's most recent delivered order
and today. Used for RFM recency scoring.
{% enddocs %}

{% docs customer_segment %}
RFM-based customer segment derived from recency of last order.
One of: active (<=30 days), at_risk (<=90 days),
lapsing (<=180 days), churned (>180 days), never_ordered.
{% enddocs %}

{% docs has_ordered %}
Boolean flag indicating whether the customer has placed
at least one delivered order.
{% enddocs %}

{% docs gross_margin_gbp %}
Gross profit in GBP. Calculated as total revenue minus
(total units sold × cost price per unit).
{% enddocs %}

{% docs gross_margin_pct %}
Gross margin as a percentage of total revenue.
Null when no sales have been recorded for the product.
{% enddocs %}

{% docs total_revenue_gbp %}
Total net revenue generated by the product across all delivered orders, in GBP.
{% enddocs %}

{% docs total_units_sold %}
Total number of units of this product sold across all delivered orders.
{% enddocs %}

{% docs avg_selling_price_gbp %}
Average unit selling price across all delivered order lines, in GBP.
{% enddocs %}
